# Chat Server with MultiThread


멀티스레드를 활용한 간단한 1:N 채팅 서버

### 명령어 목록

|명령어|설명|
|:---|:---|
|/Login|ID 입력하면 자동 로그인|
|/Logout|로그아웃|
|/Enter [arg]|매개변수로 방 이름 이력하여 방 생성 및 입장|
|/Leave|방에서 퇴장|
|/Exit|프로그램 종료|
|/RoomList|방 목록 출력|
|/UserList|입장한 방에 있는 유저 목록 출력|
